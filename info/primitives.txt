  ///Number

  https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Number/EPSILON

  Number.isFinite() - перевіряє чи кінечне число

  Number.isInteger() - чи ціле число 

  Number.isNaN()- перевіряє чи є саме число NaN
  Number.isNaN("NaN"); // false
  Number.isNaN(undefined); // false
  Number.isNaN({}); // false
  Number.isNaN("blabla"); // false
  при просто isNaN() це все true тому краще використовуємо його 

  Number.parseFloat() - повертає флоат, але не рекомендовано бо:
  Number.parseFloat('213213.31232 adasdasasd') //213213.31232
  тому краще Number('213213.31232 adasdasasd') //NaN

  Number.parseInt() - теж саме що парсфлоат але з приколом: 
  Number.parseInt(string[, radix])
  radix - це уточнення у якій системі числення стрінг
  Number.parseInt('11101101', 2) // 237
  можна використовувати саме для перенесення у інші системи числення


  усі .toExponential() .toFixed() повертають саме рядок
