transform:

  transform(X\Y): traslate(X, Y) перемішення на чогось у кудись

  transform: traslate(50%, 50%)  змішує на 50% вверх й 50% у право (відсодсоток від довжини себе)


  transform:scale(X, Y) змінення розміру 
  transform:scaleX(X) scaleY(Y) змінення розміру v2 

  transform:scale(1, 1) - стандартне значення

  transform:scale(-X, Y) ефект "перевертання" по X
  transform:scale(X, -Y) ефект "перевертання" по Y


  transform: rotate(X) повертання чогось (deg/turn/rad)

  transform: skew(X, Y) розтягання елемента по осям (deg/turn/rad)
  transform: skewX(X) skewY(Y) розтягання елемента по осям (deg/turn/rad) v2

  perspective: X - ввимкнути 3д режим для об'єкта (лише для об'єктів) + потрібно вмикати для об'єктів "батьків"
  
  !!наступні мають бути з perspective: X 
  та кожне значення НЕ МАЄ БУТИ БІЛЬШЕ ЗА X, інакше буде виїзжати за екран користувача!!

    transform:traslate3d(X,Y,D) - теж саме але 3Д
    transform:traslate3d(0,0,X) - ефект подібний до transform:scale
    

    transform:scale3d(X,Y,Z)- маштабує за трьома осями, щоб побачити Z іншими перетвореннями


    transform:rotate3d(X,Y,Z,D) - обертання у трьох вимірах, але:
      перші три значення X,Y,Z - формування вектору напрямку
      четверте значення D - градусів прокрутки
    

    transform-style: preserve-3d/flat - поведінка 3d об'єктів к батькам (говорить що потрібно моделювати (або не) з об'єкта 3д об'єкт (усі його сторони))

  
  transform-origin:top/bottom/left/right - з відки починається трансформація об'єкта
