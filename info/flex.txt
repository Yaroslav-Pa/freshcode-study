30.10.2023 - батьківський елемент флекса

  flex-direction: Y; - визачення напрямку flex 
    Y: 
    row = головна - горизонтальна, другорядна - вертикальна
    column = головна - вертикальна, другорядна - горизонтальна 

    justify-content: X; вирівнює контент ЗА ГОЛОВНОЮ ЛІНІЄЮ що визначається у flex-direction
      X:
      center -центр
      flex-end, flex-start - кінець та початок
      space-around - динамічно відштовхує усе, роботає по принципу падінга по X-Y
      space-between - розподіляє дітей по всьому простору НЕ роблячи відступів по бокам
      space-evenly - розподіляє дітей по всьому простору З ВІДСТУПАМИ по бокам
    
    align-items: X; вирівнює контент ЗА ДРУГОРЯДНОЮ ЛІНІЄЮ що визначається у flex-direction
      X: 
      stretch(за замовчуванням, якщо не вказати висоту/ширину) - розтягує усі об'єкти до макс висоти/ширини
      center -центр
      flex-end, flex-start - кінець та початок
      baseline - вирівнювання відносно першого рядка (базової лінії тексту)

    warp\nowarp - перенесення або не перенесення контенту на наступну/попередню строку

    align-content: X; вирівнювання усіх об'єктов що були перенесені від warp warp-reverse (якщо є пусте місце у висоті/ширині батька-об'єкта)
      stretch(за замовчуванням) - розтягує усі об'єкти до макс висоти/ширини
      center -центр
      flex-end, flex-start - кінець та початок
      baseline - вирівнювання відносно першого рядка (базової лінії тексту)

  !! flex-flow = flex-direction + warp\nowarp


31.10.2023 - child елемент флекса

  order - змінює візуальну позицію елемента, якщо буде однаковим пійде за порядком у html

  align-self
  
  flex-basis встановює стандартне значення, "перезаписує" усе крім min-XY max-XY
   
  flex-grow (стандарт 0)- розподіляє вільне місце між усіма дітьми флекса за встановленим
    кофіцієнтом (у тіх що не визначено кофіцієнт = 0)   
      наприклад: 1480px усього | 800px вільно | 136px розмір (5 елементів по кофіцієнту 1) => 1480 | 0 вільно| 296px розмір
    
  flex-shrink (стандарт 1) - старається розмістити усе в межах флекс контейнера
    зменьшуючи усе як хоче, розраховує усе приблизно як flex-grow
  
  flex = flex-grow + flex-shrink + flex-basis - скорочення
    flex: 2 3 150px;
